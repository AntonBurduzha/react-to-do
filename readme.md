#Simple To-Do List
###Install
1. ```npm install```

###Usage
#####Web app
1. ```npm run prod```
2. ```npm run dev```

#####Electron app
1. ```npm run build-css```
2. ```npm run electron```

#####Fixing webpack errors associated with node-sass/vendor
1. ```npm rebuild node-sass```
2. delete node_modules
3. npm install
